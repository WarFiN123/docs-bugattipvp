## Introduction

Kits are free loot that recharges. Currently, we have implemeted 2 kits: 
- The **Starter** Kit
- The **Food** Kit

![kits](https://github.com/WarFiN123/docs-bugattipvp/blob/main/contents/docs/usage/auction_house/kits.png?raw=true)

## Kits

To view the kits, run the command below:
```bash
/kits
```
