To join, you need to have the following installed:

## Minecraft (ofcourse)

<Note title="Important" type="danger">
  You need to have the **official Minecraft** installed to get in the server.
  Tlauncher won't work.
</Note>
<br />

### Supported Versions

- Bedrock Edition
- Java Edition
- Pocket Edition
- 1.16.2 - 1.20.1
