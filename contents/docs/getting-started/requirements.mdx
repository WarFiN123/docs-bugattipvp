To join, you need to have the following installed:

## Minecraft (ofcourse)

### Supported Versions

- Bedrock Edition
- Java Edition
- Pocket Edition
- 1.16.2 - 1.21.1
- Any cracked client ([TLauncher](https://tlauncher.org/)) is also supported
